## Build Systems

### Autotools

The traditional build system:

```bash
# Configure
./autogen.sh
./configure

# Build
make

# Test
make check

# Install
make install
```

### Meson

The modern build system:

```bash
# Configure
meson setup builddir

# Build
meson compile -C builddir

# Test
meson test -C builddir

# Install
meson install -C builddir
```
